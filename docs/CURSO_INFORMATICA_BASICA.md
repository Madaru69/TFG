# Curso: Alfabetización Digital y Hardware (Nivel Básico)

## Descripción General
Curso introductorio diseñado para comprender los fundamentos de un sistema informático, desde los componentes físicos hasta el uso seguro de internet.

---

## Módulo 1: ¿Qué hay dentro de un ordenador? (Hardware)

### Contenido Teórico
Un ordenador se divide en dos partes fundamentales: **Hardware** (lo que tocas) y **Software** (lo que ves).

#### Componentes Principales:
1.  **CPU (Procesador):** El cerebro. Ejecuta las órdenes. (Ej: Intel Core, AMD Ryzen).
2.  **RAM (Memoria Volátil):** La mesa de trabajo. Todo lo que haces ahora mismo está aquí. Si apagas, se borra.
3.  **Disco Duro (Almacenamiento):** El archivo. Aquí se guardan tus fotos y programas para siempre. (HDD o SSD).
4.  **Placa Base:** La carretera que conecta todo.
5.  **Periféricos:**
    *   *Entrada:* Teclado, Ratón, Micrófono.
    *   *Salida:* Monitor, Altavoces, Impresora.

### Actividad Sugerida
*   **"Diseña tu PC":** Usar el simulador "PC Building Simulator" (o una web de configuración como PCComponentes) para elegir piezas compatibles.

---

## Módulo 2: El Director de Orquesta (Sistema Operativo)

### Contenido Teórico
El Sistema Operativo (SO) es el software que habla con el hardware por ti.

#### Tipos Principales:
*   **Windows:** El más común en oficinas y hogares. Fácil de usar, pero de pago.
*   **macOS:** Exclusivo de Apple. Famoso por su diseño y estabilidad.
*   **Linux:** Gratuito y de código abierto (Open Source). Usado en servidores y superordenadores.
*   **Android/iOS:** Sistemas operativos móviles.

### Actividad Sugerida
*   **"Explorador de Archivos":** Crear una estructura de carpetas lógica (Año -> Mes -> Documentos) para organizar archivos personales.

---

## Módulo 3: Internet y Seguridad

### Contenido Teórico
Internet es una red de redes conectadas globalmente.

#### Conceptos Clave:
*   **Navegador (Browser):** El coche para viajar (Chrome, Firefox, Edge).
*   **Buscador:** El mapa (Google, Bing).
*   **La Nube:** Ordenadores de otras personas donde guardas tus datos (Google Drive, Dropbox).

#### Reglas de Oro de Ciberseguridad:
1.  **Contraseñas:** Nunca usar "123456". Usar frases: "MiPerroComePizzaEn2024!".
2.  **Phishing:** Si el banco te pide tu clave por email, es mentira.
3.  **Actualizaciones:** Siempre instalar las actualizaciones de Windows, arreglan agujeros de seguridad.

### Actividad Sugerida
*   **"Test de Contraseña Segura":** Usar una web como *HowSecureIsMyPassword* (con ejemplos falsos) para ver cuánto tardaría un hacker en adivinarla.
